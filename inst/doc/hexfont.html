Overview

This is an R data package that includes every â€œhexâ€ font file from the GNU Unifont project compressed by xz. It includes a convenience function that loads several of them as a {bittermelon} bm_font() object. GNU Unifont is a duospaced bitmap font (8x16 and 16x16 glyphs) that pretty much covers all of the official Unicode glyphs plus several of the artificial scripts in the (Under-)ConScript Unicode Registry.

Included hex font files

{hexfont} includes all the hex font files included in the GNU Unifont source code distribution. The version number is stripped from the file names in the precompiled directory and the hex fonts are all compressed by xz but other than that the hex fonts are otherwise unchanged.

hex_dir <- system.file("font", package = "hexfont")
list.files(hex_dir, pattern = ".hex.xz", recursive = TRUE)

Examples

The main function unifont() loads in several GNU Unifont hex files at the same time as a {bittermelon} bm_font() object. Arguments:

Argument Meaning Default
upper Include glyphs above the Unicode Basic Multilingual plane TRUE
jp Use Japanese version of Chinese characters FALSE
csur Include (Under-)Conscript Unicode Registry glyphs TRUE
sample Add circle to â€œCombiningâ€ characters FALSE
ucp Character vector of Unicode Code Points to only load NULL 
library("bittermelon") # remotes::install_github("trevorld/bittermelon")
##

##

library("hexfont") # remotes::install_github("trevorld/hexfont")
system.time(font <- unifont()) # Unifont is a **big** font
##    user  system elapsed 
##  52.149   0.148  52.324

## [1] "79,162"

## [1] "118.4 Mb"
# Faster to load from a cache

system.time(font <- readRDS("unifont.rds"))
